## 3. 搜索与替换

### 查找对话框 (Ctrl+F)

| 选项 | 说明 |
|------|------|
| 查找文本 | 支持普通文本、正则表达式、扩展（`\n`、`\t` 等） |

| 在选区内 | 仅在当前选区内搜索 |
| 标记行 | 将所有匹配行添加书签 |
| 高亮所有 | 用指示器（Indicator）高亮所有匹配 |
| 查找下一个 | F3 |
| 查找上一个 | Shift+F3 |

## 9. 插件系统

### 插件 API

Notepad++ 插件通过 **Win32 DLL** 实现，导出以下标准接口：

```cpp
extern "C" {
    void setInfo(NppData);          // 接收主窗口句柄
    const TCHAR* getName();         // 返回插件名称
    FuncItem* getFuncsArray(int*);  // 返回菜单项列表
    void beNotified(SCNotification*); // 接收编辑器通知
    LRESULT messageProc(UINT, WPARAM, LPARAM); // 接收消息
    BOOL isUnicode();               // 声明 Unicode 支持
}
```

### Plugin Admin（插件管理器）

| 功能 | 说明 |
|------|------|
| 安装插件 | 从在线仓库列表选择并安装 |

| 卸载插件 | 从插件目录删除 |
| 热重载 | 无需重启即可重载部分插件 |
